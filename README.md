# 人はなぜ、同じものを違う方法で楽しむのか？

商品やコンテンツの再解釈に関する研究サイト

## 概要

このサイトは「同じ商品・コンテンツでも、人によって楽しみ方や使い方が変わる」というテーマを探求するWebサイトです。実際のアンケート調査結果をもとに、人々がどのように商品を本来の目的以外で楽しむのかを紹介しています。

## 主な事例

- **明治エッセルスーパーカップ**: アイスクリーム → 食べ方比較・SNS投稿・食感体験
- **フィルムカメラ**: 記録道具 → 感性表現・ファッション・SNS演出  
- **カフェ**: 飲食店 → 勉強・作業・写真撮影・空間体験

## サイト構成

1. **Home**: メインタイトルと事例紹介
2. **Intro**: 目的と再解釈の概念説明
3. **Research**: カードゲームに関するアンケート結果
4. **Archive**: 事例アーカイブ
5. **Explore**: 詳細な事例紹介（モーダル表示）
6. **Discover**: 結論と洞察
